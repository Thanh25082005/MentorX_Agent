# Chương trình đào tạo - Trung tâm AI Academy

## Tổng quan

Trung tâm AI Academy cung cấp các chương trình đào tạo chất lượng cao trong các lĩnh vực:
- Lập trình (Python, Web, SQL, Git)
- Data Science & Analytics
- AI / Machine Learning / Deep Learning
- DevOps & Cloud
- Bảo mật thông tin

## Lộ trình học

### Lộ trình 1: Lập trình viên Python
1. Python Cơ Bản (8 tuần) → Python Nâng Cao (10 tuần) → chọn chuyên ngành

### Lộ trình 2: Data Scientist
1. Python Cơ Bản (8 tuần) → Data Science với Python (12 tuần) → Machine Learning A-Z (16 tuần)

### Lộ trình 3: AI Engineer
1. Python Cơ Bản → Python Nâng Cao → Machine Learning A-Z → Deep Learning & NLP → Generative AI & LLM

### Lộ trình 4: Full-stack Developer
1. Git & GitHub → Web Development Full-stack → SQL & Database → Docker & DevOps

## Hình thức học

- **Offline**: Tại trung tâm, tối đa 20 học viên / lớp
- **Online**: Qua Zoom, có recording
- **Hybrid**: Kết hợp offline + online

## Giảng viên

Tất cả giảng viên đều có:
- Tối thiểu 5 năm kinh nghiệm thực tế
- Đang làm việc tại các công ty công nghệ hàng đầu
- Có chứng chỉ quốc tế liên quan

## Chứng chỉ

- Hoàn thành khóa học sẽ nhận **Chứng chỉ hoàn thành** của AI Academy
- Các khóa ML/DL/GenAI có thêm **portfolio project** để bổ sung vào CV
- Trung tâm hỗ trợ thi chứng chỉ quốc tế (AWS, Google Cloud, Azure)

## Đầu ra cam kết

- Học viên hoàn thành lộ trình được hỗ trợ **giới thiệu việc làm** tại các công ty đối tác
- Có buổi **mock interview** và review CV miễn phí
- Cộng đồng alumni hỗ trợ lâu dài
